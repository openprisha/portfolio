Project: 1.1.4 -  Save the Day!

Course: PLTW Cybersecurity

Duration: September 2021

The objective of this project was to efficiently use the Cybersecurity lifecycle to fix issues of a “coworkers” computer and increase the security of their personal information and detect any potential security threats. I was instructed to take up the role to be a help desk employee for a cybersecurity company (all as a part of a fake simulation to reflect the actual workplace), and answer any questions that my customers/coworkers would ask in regards to their computer safety issues. I was also instructed to create my own personal artifact that I would use to educate my clients and others on basic cyber hygiene and personal safety rules online. As a part of the project I had to educate myself on online cyber hygiene rules, and how to go about fixing different kinds of viruses that people’s software could get affected by. By answering the questions of clients about cybersecurity questions, I was able to learn about how a cybersecurity help desk works, and learn about ways to deal with cyber issues like password vulnerabilities, email threats, and additional social media security risks. Project: 1.2.5 - It’s a Trap!

Course: PLTW Cybersecurity

Duration: September - October 2021

This project was a simulation project to replicate a security scenario of a computer that has been compromised. The scenario that I was given was that a friend who had inherited her old computer from her sister realizes that she had lost some data and fears the Computer system is having security issues. My job in this project was to use my cybersecurity skills and knowledge of the cybersecurity lifecycle to identify threats in her computer and fix them for her. Prior to the project, learned about the basic types of malware, additional security features on a browser to protect a computer (learn about antivirus software), and how to avoid being vulnerable to a malware attack. I also learned about files and processes in a computer’s desktop and how to manage them, and how to identify suspicious data (potential malware) from these processes. I ended up using my skills to create a plan to secure the safety of her computer by improving the firewall's security, discovering and deleting any malware processes that I found running in her computer, recovering lost information, and finding additional suspicious or harmful files in her computer system and eliminating them. From this project, I was able to overcome security threats from a malware attack from a victim’s computer, determine how the attack occurred, and end up securing their browser to prevent any future attacks through the usage of antivirus software.
